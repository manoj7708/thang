# Assets

This folder holds your brand assets. Two files are referenced in the code:

| File | Where it's used | What to do |
|---|---|---|
| `logo.png` | Navbar, Footer, Home hero badge, About story | Replace with your **Thangalatha Finance logo** (transparent PNG, square, ideally 512×512). |
| `owners.png` | Home hero section (the floating photo with gold glow) | Replace with the **owners' photo with the background removed** (transparent PNG, portrait orientation works best). |

Both files are currently placeholders generated by `generate_placeholders.py` at the project root — just so the dev server runs out of the box.

## How to remove the background from the owners photo

Pick any of these:

1. **remove.bg** — https://www.remove.bg — paste the photo, download the transparent PNG, save as `owners.png` in this folder.
2. **Photoshop** — Select → Subject → Refine → Delete background → Export as PNG.
3. **GIMP** — Free, Tools → Foreground Select.
4. **Python (rembg)**:
   ```bash
   pip install rembg
   rembg i owners-original.jpg owners.png
   ```

Save the final file here as `owners.png`. The code already references it — no changes needed.
